## 切割規則備註
- 碎片門檻：像素 < 40 且底部集中，下方 > 200px → 合併 32x48
- 合併方式：上方格子底部 16px + 下方格子完整 32px
- 強制合併：grid (0,13)+(1,13)、(0,14)+(1,14)、(0,15)+(1,15)
- 強制不合併：grid (3,8)、(3,10)

## 設計決策備註
- 不區分薄素材，統一用圖層堆疊
- 碰撞在同高度+同圖層內檢查
- 操作只作用於當前高度+圖層
- 儲存為 JSON 檔案匯出，組合存 localStorage
