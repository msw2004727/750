# 方塊 - 等距方塊建造器

## 專案概述
基於瀏覽器的等距 (isometric) 方塊建造器，純 HTML/Canvas/JavaScript，未來將發展為遊戲。

## 檔案結構
```
方塊/
├── isometric_block_builder.html  # 主頁面
├── style.css                     # 樣式
├── game.js                       # 遊戲邏輯
├── CLAUDE.md
├── claude-memory.md
├── 素材/
│   ├── isometric tileset/        # 第一組（115 張 32x32）
│   │   └── separated images/
│   ├── isometric_jumpstart_v230311/  # 第二組（132 張，32x32 + 32x48）
│   │   ├── separated/
│   │   └── tile_log.txt          # 切割日誌
│   └── 3232iso/                  # 第三組（143 張 32x32，具名檔案）
└── 臨時/
```

## 技術參數
- TILE = 40, 等距角度 = 30°
- CUBE_H = TILE * 0.8（一格高度）
- 貼圖對齊方塊底部，透明空間在上方
- 像素風格：imageSmoothingEnabled = false
- 圖片路徑含中文（素材），URL 編碼為 %E7%B4%A0%E6%9D%90

## 座標系統
- **高度** (gz)：-5 到 +5，方塊垂直堆疊位置
- **圖層** (layer)：0 到 5，同高度同位置的重疊深度
- 碰撞只在**同高度 + 同圖層**內檢查
- 操作（拖曳、刪除）只作用於當前高度 + 當前圖層的物件
- hitTest 只偵測當前高度 + 圖層，避免誤選

## 素材尺寸規則
- 32x32：一格，佔 1 層高度
- 32x48：一格半，佔 2 層高度（當前層 + 上方層，srcH > 32 判定）
- drawCube 以寬度為基準等比縮放，保持原始寬高比

## 功能清單
- 三來源分頁（原始素材 / Jumpstart / 3232iso），每來源有子分類
- 素材按鈕顯示編號
- 高度 ▲▼ + 圖層 ▲▼ 切換
- 座標顯示開關
- Flood fill 拖曳限制（被圍牆困住只能在內部移動）
- 儲存/載入場景（JSON 檔案匯出匯入）
- 自訂組合（存選取 → 命名 → 一鍵重複放置）
- 組合存在 localStorage，持久保留
- 空白處拖曳平移視角、滾輪縮放
- 雙擊/右鍵刪除方塊

## 切割規則（jumpstart 素材）
- 基礎格子：32x32（16x10 grid）
- 碎片判定：像素數 < 40 且內容集中在底部，下方格子 > 200px → 合併為 32x48
- 合併方式：上方格子底部 16px + 下方格子完整 32px
- 強制合併：grid (0,13)+(1,13)、(0,14)+(1,14)、(0,15)+(1,15)
- 強制不合併：grid (3,8)、(3,10)（獨立小裝飾）
